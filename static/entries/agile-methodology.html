Introduction

This was a workshop led by Isabel Cousins all about the agile methodology and how to use it effectively when managing a project. It consisted of us splitting off into groups to design a rooftop terrace and present our ideas to the rest of the cohort, allowing us to practice group management using agile.

Overview

As stated, our overall aim was to put into practice the agile methodology in order to create a design for a sustainable rooftop terrace. This would allow us to improve on the essential skills within project management such as staying in scope and working within the sprints.


Challenges

The main challenge was that we were all new to the agile methodology and had to learn how to use it effectively. It was quite challenging to get used to the different roles of the group, however, we were able to work together efficiently to create a successful solution.
